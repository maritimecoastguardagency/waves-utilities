### Usage

To add to a project:

`gem "waves-utilities", git: "https://github.com/tobyprivett/waves-utilities.git"`

To open a command prompt:

`bin/console`

#### Countries

`WavesUtilities::Country.all`

returns a list as:
`["GB", "UNITED_KINGDOM"]`

#### Nationalities

`WavesUtilities::Nationality.all`

returns a list as:
`["UNITED KINGDOM"]`


#### Vessel Types

`WavesUtilities::VesselTypes.all`

returns a list as:
`["BARGE"]`
